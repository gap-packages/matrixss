Schreier-Sims algorithm for matrix groups

This is the homepage of a project aimed at implementing an efficient version of the Schreier-Sims algorithm for matrix groups. The implementation is being made in GAP.

GAP already includes implementations of the algorithm, but those are for permutation groups. It is possible to use these implementations for matrix groups, but since matrix groups are "larger" than permutation groups the algorithms that works for permutation groups are not in general efficient for matrix groups, and Schreier-Sims algorithm is one example.

Project development summary

Information about the Schreier-Sims algorithm can be found in the following books (an in lots of other places)

  • G. Butler, Fundamental Algorithms for Permutation groups
  • A. Seress, Permutation group algorithms
  • A. Cohen, H. Cuypers, H. Sterk (editors), Some Tapas of Computer Algebra
